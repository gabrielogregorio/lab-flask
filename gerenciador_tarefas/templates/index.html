REST API COM FLASK

CRUD API FLASK - Gerenciador de tarefas

Está é uma versão de estudos com REST API para um gerenciador de tarefas.

Para usa-lo é necessário seguia a tabela de referências ao lado.

Função TIPO URL CORPO
Adicionar tarefas POST http://127.0.0.1:5000/add { "responsável": "", "status": "","tarefa": "" }
Listar todas as tarefas GET http://127.0.0.1:5000/task
Verificar tarefa GET http://127.0.0.1:5000/task/1
Editar tarefas PUT http://127.0.0.1:5000/task/1 {"responsável": "", "status":"", "tarefa": "" }
Deletar tarefas DELETE http://127.0.0.1:5000/task/1
